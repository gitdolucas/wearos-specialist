# Security

This repository contains documentation and guidance (markdown, templates, and configuration). It does not run or distribute executable code or handle user data.

## Reporting a vulnerability

If you believe you have found a security issue related to this project (for example, malicious or unsafe guidance that could lead to vulnerable apps, or a problem with the distribution or integrity of the content), please report it:

1. **Open a private report** by contacting the maintainers through the repository’s preferred contact method (e.g. GitHub Security Advisories, or the contact listed in the repo description).
2. Include a clear description of the issue and steps to reproduce, if applicable.
3. We will acknowledge the report and respond with next steps. We do not have a formal bug bounty program.

We take reports seriously and will address valid concerns as quickly as we can. Thank you for helping keep the community safe.
